Visual Studio Code – Általános

Gyors navigáció és beállítások megnyitása:

Ctrl Shift P     # Parancspaletta megnyitása
Ctrl P           # Gyors megnyitás, ugrás fájlra
Ctrl Shift N     # Új ablak/instance
Ctrl Shift W     # Ablak/instance bezárása
Ctrl ,           # Felhasználói beállítások
Ctrl K Ctrl S    # Billentyűparancsok megnyitása

VSC – Alap szerkesztés (1)

Sorok vágása, másolása, mozgatása és beszúrása:

Ctrl X             # Sor kivágása (üres kijelölésnél)
Ctrl C             # Sor másolása (üres kijelölésnél)
Alt Fel/Le         # Sor mozgatása fel/le
Shift Alt Fel/Le   # Sor másolása fel/le
Ctrl Shift K       # Sor törlése
Ctrl Enter         # Új sor beszúrása alá
Ctrl Shift Enter   # Új sor beszúrása fölé
Ctrl Shift \       # Ugrás a párzárójelhez
Ctrl ]/[           # Behúzás / behúzás visszavonása

VSC – Alap szerkesztés (2)

Görgetés, kódhajtogatás és kommentek kezelése:

Home/End           # Ugrás a sor elejére/végére
Ctrl Home          # Ugrás a fájl elejére
Ctrl End           # Ugrás a fájl végére
Ctrl Fel/Le        # Görgetés soronként fel/le
Alt PgUp/PgDown    # Görgetés oldalanként fel/le
Ctrl Shift [       # Régió összecsukása
Ctrl Shift ]       # Régió kinyitása
Ctrl K Ctrl [      # Minden alrégió összecsukása
Ctrl K Ctrl ]      # Minden alrégió kinyitása
Ctrl K Ctrl 0      # Minden régió összecsukása
Ctrl K Ctrl J      # Minden régió kinyitása
Ctrl K Ctrl C      # Sor komment hozzáadása
Ctrl K Ctrl U      # Sor komment eltávolítása
Ctrl /             # Sor komment ki/be
Shift Alt A        # Blokk komment ki/be
Alt Z              # Sortörés (word wrap) ki/be

Gyors ugrás sorra, fájlra és szimbólumokra, hibák kezelése:

Ctrl T             # Minden szimbólum megjelenítése
Ctrl G             # Ugrás adott sorra...
Ctrl P             # Ugrás fájlra...
Ctrl Shift O       # Ugrás szimbólumra...
Ctrl Shift M       # Hibák/figyelmeztetések panel
F8                 # Következő hiba vagy figyelmeztetés
Shift F8           # Előző hiba vagy figyelmeztetés
Ctrl Shift Tab     # Editor csoport előzmények
Alt Balra/Jobbra   # Vissza/Előre
Ctrl M             # Tab fókusz mozgatás ki/be

VSC – Keresés és csere

Találatok keresése, kiválasztása és cseréje:

Ctrl F             # Keresés
Ctrl H             # Csere
F3                 # Következő találat
Shift F3           # Előző találat
Alt Enter          # Minden találat kijelölése
Ctrl D             # Kijelölés hozzáadása a következő találathoz
Ctrl K Ctrl D      # Utolsó kijelölés mozgatása a következő találathoz
Alt C/R/W          # Kis/nagybetű / regex / teljes szó ki/be

VSC – Több kurzor és kijelölés

Több kurzor kezelése és kijelölések finomítása:

Alt Kattintás              # Kurzor beszúrása
Ctrl Alt Fel/Le             # Kurzor beszúrása fölé/alá
Ctrl U                      # Utolsó kurzor művelet visszavonása
Shift Alt I                 # Kurzor minden kijelölt sor végére
Ctrl I                      # Aktuális sor kijelölése
Ctrl Shift L                # Minden előfordulás kijelölése
Ctrl F2                     # Szó összes előfordulásának kijelölése
Shift Alt Jobbra            # Kijelölés bővítése
Shift Alt Balra             # Kijelölés szűkítése
Shift Alt (egér húzás)      # Oszlop (doboz) kijelölés
Ctrl Shift Alt (nyilak)     # Oszlop kijelölés billentyűkkel
Ctrl Shift Alt PgUp/PgDown  # Oszlop kijelölés oldalanként
