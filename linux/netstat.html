Netstat – Listening portok

Az alábbi parancsokkal megjeleníthetők a különböző típusú figyelő portok:

netstat -ltunp    # Minden figyelő port  
netstat -ltn      # Figyelő TCP portok  
netstat -lun      # Figyelő UDP portok  
netstat -lx       # Figyelő Unix portok

Netstat – Aktív kapcsolatok

Az alábbi parancsokkal megjeleníthetők az aktív hálózati kapcsolatok:

netstat -a     # Minden kapcsolat  
netstat -at    # TCP kapcsolatok  
netstat -au    # UDP kapcsolatok

Netstat – Statisztikák

Az alábbi parancsokkal megjeleníthetők a protokoll statisztikák:

netstat -s     # Összes statisztika  
netstat -st    # TCP statisztika  
netstat -su    # UDP statisztika

Netstat – Hálózati interfészek

Az alábbi parancsokkal megjeleníthetők az interfészek és azok állapota:

netstat -i     # Interfészek listája  
netstat -ie    # Részletes interfész információ

Netstat – Routing tábla

Az alábbi parancsokkal megjeleníthető az útválasztási tábla:

netstat -r     # Routing tábla  
netstat -rn    # Routing tábla, névfeloldás nélkül

Netstat – Port 80 kapcsolatok

Megjeleníti az összes kapcsolatot, amely a 80-as portot használja:

netstat -anp | grep :80

Netstat – Súgó

Megjeleníti a netstat parancs súgóját és opcióit:

netstat -h
