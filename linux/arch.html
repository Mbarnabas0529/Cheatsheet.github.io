Arch Linux – pacman napló

napló megtekintése:
/var/log/pacman.log

Arch Linux – pacman frissítés

# pacman -Syu

Arch Linux – pacman listázás

telepített csomagok
# pacman -Q

nem szükséges csomagok
# pacman -Qdtq

Arch Linux – pacman keresés

telepített csomagok keresése
# pacman -Qs 

csomagok keresése
# pacman -Ss

Arch Linux – pacman telepítés/eltávolítás

csomag telepítése
# pacman -S 

csomag eltávolítása függőségekkel
# pacman -Rns 

cache tisztítása
# pacman -Sc

Arch Linux – yaourt

megegyezik a pacman parancsokkal

Arch Linux – systemd alapok

unit fájlok:
/usr/lib/systemd/system/
/etc/systemd/system/

futó unitok listázása
$ systemctl

állapot ellenőrzése
$ systemctl status

Arch Linux – systemd szolgáltatások

szolgáltatás indítása/leállítása
# systemctl (start|stop) 

szolgáltatás engedélyezése/letiltása induláskor
# systemctl (enable|disable) 

systemd újratöltése
# systemctl daemon-reload

Arch Linux – AUR kézi telepítés

tárolók frissítése
# pacman -Sy

csomag letöltése
$ curl -O 

csomag kicsomagolása
$ tar xzvf 

belépés a könyvtárba
$ cd 

fordítás és telepítés
$ makepkg -si

Arch Linux – Java környezetek

állapot ellenőrzése
$ archlinux-java status

alapértelmezett verzió beállítása
# archlinux-java set

Arch Linux – Fájlrendszer és lemezkezelés

Lemezhasználat ellenőrzése
df -h

Könyvtár méretének ellenőrzése
du -sh 

Lemez csatolása
sudo mount /dev/ /mnt

Lemez leválasztása
sudo umount /mnt

Meghajtók és partíciók listázása
lsblk

Meghajtó formázása ext4-re
sudo mkfs.ext4 /dev/

Arch Linux – Felhasználókezelés

Új felhasználó hozzáadása
sudo useradd -m -G wheel -s /bin/bash 

Jelszó beállítása
sudo passwd 

Átváltás másik felhasználóra
su - 

Felhasználó törlése
sudo userdel -r

Arch Linux – Hálózatkezelés

Aktív kapcsolatok megjelenítése
ip a

Interfész engedélyezése/letiltása
sudo ip link set  up
sudo ip link set  down

Hálózati kapcsolat tesztelése
ping 

NetworkManager újraindítása
sudo systemctl restart NetworkManager

Arch Linux – Szolgáltatások kezelése

Szolgáltatás állapotának ellenőrzése
systemctl status 

Indítás / Leállítás / Újraindítás
sudo systemctl start 
sudo systemctl stop 
sudo systemctl restart 

Engedélyezés induláskor
sudo systemctl enable 

Letiltás induláskor
sudo systemctl disable

Arch Linux – Kernel és boot kezelés

Telepített kernelek listázása
ls /boot/vmlinuz*

GRUB konfiguráció frissítése
sudo grub-mkconfig -o /boot/grub/grub.cfg

Új kernel telepítése
sudo pacman -S linux-lts

Régi kernel eltávolítása
sudo pacman -R linux-

Arch Linux – Hibakeresés

Rendszerlogok megtekintése
journalctl -xe

Boot logok megtekintése
dmesg | less

Hibás systemd unitok listázása
systemctl --failed

Arch Linux – Hasznos parancsok

Csomag cache tisztítása
sudo pacman -Sc

Arch Linux rendszerinformáció
neofetch

Mirrorlist generálása
sudo reflector --latest 20 --sort rate --save /etc/pacman.d/mirrorlist

pacman konfiguráció szerkesztése
sudo nano /etc/pacman.conf
